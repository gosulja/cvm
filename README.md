# cvm (cat virtual machine)
A register based bytecode virtual machine compiler.

# features
* Mathematical expressions such as `1+2` or `(5*5) - 10 / 2`.

# building
```
git clone https://github.com/gosulja/cvm
```
Either run:
```
g++ -Wall -g -o cvm main.cpp
```
or
```
make
```
then:
```
./cvm
```
and enjoy!
